{% translate "Call for Workshops & Tutorials" %}

{% blocktranslate %} In the context of the 1st African Cities Conference, we are launching a call for proposals for the organization of workshop sessions, creative or reflective events and tutorials during ACC 2023. These types of events at ACC 2023 will provide a platform to discuss a specific topic related to African cities with an audience specifically interested in that topic in an informal setting (as opposed to the main conference). {% endblocktranslate %}

{% blocktranslate %} We invite proposals for events that promote discussion, interdisciplinary exchange and interaction, scientific discourse, and creative and critical thinking, rather than simply being mini-conferences. In order to vary and diversify the activities of ACC 2023, we offer potential organizers of such events flexibility in the formats that best suit the objectives of their proposal. We also welcome proposals from research communities that are not always featured in African cities events and conferences so that the conference is not limited to typical researchers. {% endblocktranslate %}

{% blocktranslate %} So in this category of events, we hope to welcome three types of contributions: traditional workshops, creative events and tutorials, each with its own particularities as described below. {% endblocktranslate %}

{% translate "Workshops traditionnels" %}

{% blocktranslate %} Traditional workshops generally correspond to the topics and challenges of the general call for papers of the main conference. In a traditional workshop, organizers may consider a program consisting of a set of invited talks around a certain theme (with or without paper presentations). {% endblocktranslate %}

{% translate "Creative, Interactive or Thinking Events" %}

{% blocktranslate %} In the spirit of reflection and community building, we invite academics from all disciplines and individuals representing different communities of practice (including journalism, management, health, advocacy, activism, education, art, law, and politics) to contribute to creative and reflective events related to the African Cities Conference 2023. {% endblocktranslate %}

{% blocktranslate %} The format of these events is flexible. Alternative formats include, but are not limited to: {% endblocktranslate %}

  • {% translate "Speeches, Presentations, Urban Quick Talks" %}
  • {% translate "Discussions or brainstorming sessions" %}
  • {% translate "Discussions or brainstorming sessions" %}
  • {% translate "Unconference" %}
  • {% translate "Interactive workshop" %}
  • {% translate "Urban Art Exhibition or other Urban Art Intervention" %}
{% translate "Tutorials" %}

{% blocktranslate %} In this category of event, unlike traditional workshops, creative events, tutorials are intended for reasonably well established topics. However, given that topics on the challenges of African cities are emerging research topics, we hope to welcome tutorials here that link well-established topics from other disciplines to African city topics. We encourage tutorials to be interactive in order to engage the audience. {% endblocktranslate %}

{% translate "Submission Proposal Guidelines" %}

{% blocktranslate %} Proposals should be submitted via Easychair: https://easychair.org/my/conference?conf=hhai2023. {% endblocktranslate %}

{% blocktranslate %} Your proposal should not exceed 8 pages (excluding references) in IOS format, and should include the following information: {% endblocktranslate %}

  1. {% blocktranslate %} Title and summary of the workshop or tutorial. {% endblocktranslate %}
  2. {% blocktranslate %} Name, affiliation and short biography of each organizer. {% endblocktranslate %}
    • {% blocktranslate %} We recommend at least two organizers who are familiar with the field and have complementary skills. {% endblocktranslate %}
    • {% blocktranslate %} In addition to the primary organizers, workshops may involve an additional organization, such as a program committee, to evaluate contributions. {% endblocktranslate %}
  3. {% blocktranslate %} The topics and questions that the workshop or tutorial will address. {% endblocktranslate %}
  4. {% blocktranslate %} The expected duration of the workshop or tutorial (half day or full day). {% endblocktranslate %}
  5. {% blocktranslate %} A link to the previous iteration of this workshop or tutorial, if applicable. {% endblocktranslate %}
    • {% blocktranslate %} You can also discuss other closely related workshops. {% endblocktranslate %}
  6. {% blocktranslate %} A brief description of the format of the workshop or tutorial, e.g., the combination of events such as paper presentations, invited talks, panels, and general discussions. {% endblocktranslate %}
  7. {% blocktranslate %} Size of the target audience. {% endblocktranslate %}
  8. {% blocktranslate %} Special requirements, where applicable. {% endblocktranslate %}
    • {% blocktranslate %} Include a note if you have special requirements for your session that are not addressed elsewhere. {% endblocktranslate %}
  9. {% blocktranslate %} Support material, where applicable. {% endblocktranslate %}
    • {% blocktranslate %} If you have additional materials that would support your proposal, such as a video example of the animation, a website, or written workbooks, you can also include that in your proposal. {% endblocktranslate %}
{% translate "Evaluation criteria for proposals" %}
  • {% blocktranslate %} Pertinence: Relationship and pertinence to the HHAI conference objectives and the objectives listed in this particular call. {% endblocktranslate %}
  • {% blocktranslate %} Scientific quality: The potential of the proposal to generate stimulating discussions and useful results on scientific problems related to HHAI. {% endblocktranslate %}
  • {% blocktranslate %} Organisation: Is the plan clear, appropriate and complete? {% endblocktranslate %}
  • {% blocktranslate %} Diversity and inclusion: Reflection on the detailed approach to ensure engagement and participation of a broad audience. Diversity of background of workshop organizers and intended presenters/contributors. {% endblocktranslate %}
{% translate "Proceedings of the workshops and tutorials" %}

{% blocktranslate %} Each workshop, on its own, may consider archiving workshop contributions and documenting the ideas generated during the workshop. If there is sufficient interest, we may archive (with an opt-out option) all contributions to the ACC 2023 workshop, along with the workshop and tutorial summaries, as a combined CEUR Workshop Proceeding. {% endblocktranslate %}
